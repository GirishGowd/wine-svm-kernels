# SVM Kernel Comparison using the Wine Dataset

This project compares Linear, Polynomial, and RBF kernels using Support Vector Machines.

## Dataset
The Wine dataset is included in this repository and loaded using pandas.

## Files
- svm_kernels_wine.py
- wine.data
- requirements.txt
- SVM_Kernel_Comparison_Wine_Dataset_Report.pdf

## How to run
pip install -r requirements.txt
python svm_kernels_wine.py

## Output
- PCA visualization
- Decision boundary plots
- Accuracy comparison
